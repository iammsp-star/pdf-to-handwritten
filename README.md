# PDF to Handwritten - GitHub UI

This is a modern, minimalist web dashboard for converting PDFs to handwritten notes.

## Live Website
🌍 [Visit the Web App](https://iammsp-star.github.io/pdf-to-handwritten/)

## Features
- **Minimalist Aesthetic**: Soft shadows, rounded corners, uses the Inter font for a Notion/Google Docs look. Included Light and Dark modes.
- **Two-Column Split Layout**: Input and extraction on the left, settings and preview on the right.
- **Settings**: Handwriting style dropdown, paper background dropdown, color swatches (blue/black), and the "Human Imperfections" slider.

## Setup
Simply clone this repository and open `index.html` in your browser. No build steps required.
